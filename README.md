# DrawX (WIP)

DrawX is a real-time collaborative whiteboard inspired by Excalidraw.  
It allows multiple users to draw and chat together on the same canvas using WebSockets.

!  This project is currently a work in progress


##  Planned Features

- Realtime collaborative drawing
- WebSocket-based room system
- Authenticated users
- Cursor & presence sync
- Chat inside rooms
- Undo / redo
- Export boards as images


##  Tech Stack

- Next.js (Frontend)
- Node.js + WebSocket (Backend)
- Prisma + PostgreSQL
- Turborepo + pnpm
- Zod for validation
